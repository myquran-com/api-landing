# API Muslim Landing Page

Ini adalah halaman landing page sederhana untuk **API Muslim** (myQuran.com).
Halaman ini berfungsi sebagai gerbang utama (gateway) untuk mengarahkan pengguna
ke dokumentasi API versi 2 dan versi 3, serta kanal diskusi.

## Fitur Utama

- **Navigasi Dokumentasi**: Link langsung ke dokumentasi API v2 dan v3.
- **Informasi Kontak**: Link ke grup Telegram @ApiMuslim untuk saran dan
  diskusi.
- **Ringan**: Hanya menggunakan HTML, CSS, dan sedikit JavaScript untuk tahun
  copyright otomatis.

## Struktur File

- `index.html`: Kode utama halaman landing.
- `img/`: Folder aset gambar.
- `bg-6-full.jpg`: Gambar background.
